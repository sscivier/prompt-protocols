# Contributing to Prompt Protocols

We welcome contributions! This guide will help you get started.

## How to Contribute

### Suggesting Improvements
- Open an [Issue](https://github.com/sscivier/prompt-protocols/issues) describing:
  - What template/document needs improvement
  - What's unclear or could be better
  - Your suggested change

### Submitting Changes

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b improve-template-name`
3. **Make your changes**: Edit templates, add examples, improve documentation
4. **Test your changes**: Verify formatting, links, and clarity
5. **Commit with clear messages**: `git commit -m "Improve domain research template clarity"`
6. **Push to your fork**: `git push origin improve-template-name`
7. **Open a Pull Request**: Describe what you changed and why

### What We're Looking For

- **Template improvements**: Clearer prompts, better examples, additional use cases
- **Domain-specific examples**: Examples from other scientific fields
- **Documentation fixes**: Typos, broken links, unclear instructions
- **Assessment criteria refinements**: Better evaluation guidelines

### Style Guidelines

- Use clear, concise language
- Provide concrete examples
- Maintain consistent formatting with existing templates
- Follow Markdown best practices

## Questions?

Open a [discussion](https://github.com/sscivier/prompt-protocols/discussions).
