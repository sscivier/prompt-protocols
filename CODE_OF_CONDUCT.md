# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inclusive environment for all contributors to Prompt Protocols. We value respectful collaboration from researchers across all scientific disciplines and experience levels.

## Standards

**Positive behaviors we encourage:**

* Being respectful of differing viewpoints, experiences, and scientific domains
* Providing constructive feedback on templates and documentation
* Showing empathy and openness to learning from others
* Acknowledging contributions appropriately

**Unacceptable behaviors:**

* Harassment, discriminatory language, or personal attacks
* Trolling, insulting comments, or deliberate disruption
* Dismissing others' scientific domains or coding experience levels
* Any conduct that would be inappropriate in a professional setting

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, discussions, and related communications.

## Enforcement

Project maintainers will take appropriate action in response to unacceptable behavior, including removing, editing, or rejecting contributions that violate this Code of Conduct.

## Reporting

To report unacceptable behavior, contact the project maintainer directly through GitHub or open a private issue. All reports will be reviewed confidentially.
